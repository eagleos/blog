еңЁWindows 10дёҠдҪҝз”ЁWSLе®үиЈ…Centos

еҺҹеҲӣ дҪңиҖ…пјҡ з«Ҙе»әе№і 2020-11-03

рҹ’ҒрҹҸ»вҖҚвҷӮпёҸгҖҠд»Ҡж—ҘеӨҙжқЎгҖӢ 2020-11-03 гҖҠеңЁWindows 10дёҠдҪҝз”ЁWSLе®үиЈ…CentosгҖӢ з«Ҙе»әе№і

йҖҡиҝҮжң¬ж–ҮпјҢжӮЁе°ҶиҺ·еҫ—дёҖдёӘз®ҖеҚ•зҡ„ж–№ејҸеңЁWindows 10 WSLдёҠжүӢе·Ҙе®үиЈ…CentOS пјҢд»ҘдҫҝеӯҰд№ дҪҝз”ЁRHELзҡ„ RPM жҲ–YUM зӯүе‘Ҫд»ӨвҖҰвҖҰ

WSL(Windows Subsystem for Linux)дёәз”ЁжҲ·жҸҗдҫӣдәҶйўҮе…·еҗёеј•еҠӣзҡ„еүҚжІҝеҠҹиғҪгҖӮеӣ дёәWSLеҸҜд»Ҙи®©жӮЁдёҚйңҖиҰҒе®үиЈ…иҜёеҰӮVirtualBoxд№Ӣзұ»зҡ„иҷҡжӢҹеҢ–иҪҜд»¶пјҢе°ұеҸҜд»ҘзӣҙжҺҘеңЁWindows 10зі»з»ҹдёӯеӯҰд№ дҪҝз”ЁLinuxе‘Ҫд»ӨгҖӮйҖӮз”ЁдәҺLinuxзҡ„Windowsеӯҗзі»з»ҹйҖҡеёёиў«з®Җз§°дёәWSLпјҢжҳҜдёҖдёӘеңЁWindows 10е’ҢWindows Server 2019дёҠиғҪеӨҹиҝҗиЎҢеҺҹз”ҹLinuxдәҢиҝӣеҲ¶еҸҜжү§иЎҢж–Үд»¶пјҲELFж јејҸпјүзҡ„е…је®№еұӮгҖӮ

йҖҡеёёжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮMicrosoft storeе®үиЈ…зү№е®ҡLinuxеҸ‘иЎҢзүҲпјҢеҫ®иҪҜе•Ҷеә—жҸҗдҫӣдәҶдёҖдәӣйў„зј–иҜ‘зҡ„иҪ»йҮҸзә§й•ңеғҸпјҢзӣ®еүҚеҸҜдҫӣйҖүжӢ©зҡ„жңүпјҡ UbuntuгҖҒKali LinuxгҖҒDebianгҖҒAlpinLinuxгҖҒOpenSUSE е’ҢSUSE Enterprise ServerзӯүеҸ‘иЎҢзүҲгҖӮеӣ жӯӨпјҢеҰӮжһңжӮЁжғіе®үиЈ…CentOS/RHEL (Red Hat Enterprise) жҲ–иҖ…Arch Linux пјҢжӮЁдёҚеҫ—дёҚжүӢе·Ҙе®үиЈ…е®ғд»¬гҖӮйӮЈд№ҲпјҢжҲ‘д»¬иҜҘжҖҺд№Ҳж“ҚдҪңе‘ўпјҹ

еҰӮдҪ•еңЁWSLдёҠе®үиЈ… CentOS 7

жіЁж„ҸпјҡејҖе§Ӣе®үиЈ…еүҚпјҢжӮЁеҝ…йЎ»е…ҲеңЁWindows 10дёӯеҗҜз”ЁWSLпјҢеҰӮжһңжӮЁдёҚжё…жҘҡжҖҺд№Ҳж“ҚдҪңпјҢиҜ·жҹҘйҳ…еҸӮиҖғз¬”иҖ…д№ӢеүҚзҡ„ж–Үз« пјҡ

жј«и°ҲеңЁWindows Server 2019дёӯе®үиЈ…дҪҝз”ЁWSL

1: дёӢиҪҪCentOS WSL

йҰ–е…ҲжҲ‘д»¬йңҖиҰҒдёӢиҪҪдёҖдёӘеҸҜз”ЁдәҺWSLзҡ„CentOSй•ңеғҸпјҢиҜҘй•ңеғҸеә”иҜҘжҳҜдёҖдёӘз»ҸиҝҮеҺӢзј©зҡ„пјҢдё”еҢ…еҗ«дәҶrootfsеҸҠе…¶е®ғж–Үд»¶гҖӮе№ёиҝҗең°пјҢGithubжңүзүӣдәәжҸҗдҫӣдәҶдёҖдёӘејҖжәҗзҡ„еҸҜиҝҗиЎҢеңЁWSLдёҠзҡ„CentOSе®һдҫӢпјҢең°еқҖдёәпјҡhttps://github.com/yuk7/CentWSL пјҢе…¶жңҖж–°зүҲжң¬еҹәдәҺзӣ®еүҚжөҒиЎҢзҡ„зүҲжң¬CentOS 7.6гҖӮеҰӮдёӢеӣҫпјҡ

2: и§ЈеҺӢWSL CentOS 7.x

д»ҺдёҠйқўGithubең°еқҖдёӢиҪҪеҜ№еә”зҡ„еҺӢзј©ж–Үд»¶пјҢеҸій”®зӮ№еҮ»е№¶и§ЈеҺӢд№ӢпјҢи§ЈеҺӢеҗҺж–Үд»¶еӨ§е°Ҹ300MBе·ҰеҸігҖӮеҰӮдёӢпјҢжҲ‘д»¬з”ЁWinRARи§ЈеҺӢпјҢеҪ“з„¶жӮЁд№ҹеҸҜд»Ҙз”Ёзі»з»ҹй»ҳи®Өи§ЈеҺӢзЁӢеәҸжҲ–е…¶е®ғи§ЈеҺӢзј©е·Ҙе…·гҖӮ

3: еңЁWSLдёӯиҝҗиЎҢCentOS.exe е№¶е®үиЈ…

и§ЈеҺӢеҗҺпјҢжӮЁе°ҶеңЁзӣ®ж Үзӣ®еҪ•дёӯзңӢеҲ°2дёӘж–Үд»¶пјҡrootfs.tar.gzе’ҢCentOS.exeгҖӮжҲ‘д»¬йңҖиҰҒиҝҗиЎҢCentOS.exeпјҢд»Ҙдҫҝи§ЈеҺӢе…¶дёӯзҡ„ж–Үд»¶е№¶жіЁеҶҢеҲ°WSLгҖӮеҸій”®зӮ№еҮ»е№¶д»Ҙз®ЎзҗҶе‘ҳиә«д»ҪиҝҗиЎҢпјҡ

4: е®ҢжҲҗе®үиЈ…

зЁҚзӯүдёҖдәӣж—¶й—ҙпјҢзі»з»ҹе°ҶиҮӘеҠЁиҝӣиЎҢеҲқе§ӢеҢ–пјҢе®ҢжҲҗеҗҺжӮЁе°ҶзңӢеҲ°еҰӮдёӢеӣҫжүҖзӨәж¶ҲжҒҜпјҢжӯӨж—¶еҸҜжҢүд»»ж„Ҹй”®йҖҖеҮәе®үиЈ…гҖӮ

еңЁWSLдёҠеҗҜеҠЁCentOS 7

иҰҒеңЁWSLдёҠеҗҜеҠЁиҝҗиЎҢCentosпјҢиҝҷж—¶жӮЁиҝ”еӣһи§ЈеҺӢеҗҺзҡ„зӣ®еҪ•пјҢжӮЁе°ҶзңӢеҲ°еӨҡеҮәдәҶ2дёӘзӣ®еҪ•пјҢеҰӮдёӢеӣҫпјҡ

дёҚз”ЁзҗҶдјҡпјҢжӯӨж—¶еҶҚж¬ЎеҸіеҮ»е№¶д»Ҙз®ЎзҗҶе‘ҳиә«д»ҪиҝҗиЎҢCentOS.exeпјҢеҚіжҲҗеҠҹзҷ»еҪ•зі»з»ҹдәҶпјҢеҰӮдёӢеӣҫпјҡ

еңЁWSLдёҠжӣҙж–°Centos 7

зҺ°еңЁпјҢжӮЁеҸҜд»ҘйҡҸеҝғжүҖж¬ІеңЁCentOSдёҠиҝҗиЎҢжӮЁжғіиҰҒжү§иЎҢзҡ„е‘Ҫд»ӨдәҶгҖӮз„¶иҖҢпјҢдёәдәҶжЈҖйӘҢзі»з»ҹеҸҜз”ЁжҖ§пјҢжҲ‘д»¬жңҖеҘҪдҪҝз”Ёyumе‘Ҫд»ӨжөӢиҜ•дёӢжӣҙж–°зі»з»ҹпјҡ

yum -y update

еңЁWSLдёҠеҚёиҪҪCentOS 7

еҰӮжһңжӮЁеӣ дёәжҹҗз§ҚеҺҹеӣ дёҚжғідҪҝз”ЁдәҶпјҢеҸҜд»ҘдҪҝз”ЁдёӢйқўе‘Ҫд»ӨеңЁWSLдёҠеҚёиҪҪCentOSпјҡ

еҪ“жӮЁеңЁдҪҝз”ЁCentOSиҝҮзЁӢдёӯйҒӯйҒҮдёҖдәӣй”ҷиҜҜжғійҮҚиЈ…пјҢжҲ–иҖ…жӮЁдёҚе°ҸеҝғеҲ йҷӨдәҶи§ЈеҺӢзј©зҡ„зӣ®еҪ•йңҖиҰҒйҮҚиЈ…ж—¶пјҢжҲ‘д»¬йңҖиҰҒе…ҲеҚёиҪҪд№Ӣпјҡ
